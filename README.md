CODSOFT AI PROJECTS

This repository contains **3 interactive AI-based projects** built using Python and Streamlit:

1.  ARIA Rule-Based Chatbot
2.  CinemaScout Movie Recommendation System
3.  Tic-Tac-Toe AI (Minimax Algorithm)

---

##  1. ARIA Chatbot

A stylish rule-based chatbot that responds using pattern matching.

###  Features

* Greeting & conversation support
* Joke generator 
* Time & date responses 
* Basic math solving 
* Smart UI with chat bubbles

---

##  2. CinemaScout (Recommendation System)

An AI-powered movie recommendation system using TF-IDF and cosine similarity.

###  Features

* Recommend similar movies
* Filter by genre
* AI-based query answering
* Clean Netflix-like UI

---

##  3. Tic Tac Toe AI

A game built with the Minimax algorithm where you can play against AI.

###  Features

* Play vs AI 🤖
* 2-player mode 👥
* Smart unbeatable AI (Minimax)
* Score tracking system

# ⚙️ Installation & Setup

## 1️ Clone the Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

---

## 2️ Install Requirements

```bash
pip install streamlit pandas scikit-learn
```

(Optional for AI feature in Movie App)

```bash
pip install anthropic
```

---

##  How to Run Each Project


```bash
streamlit run filename.py
```

---

###  Run Chatbot

```bash
streamlit run chatbot.py
```

---

###  Run Recommendation System

```bash
streamlit run recommendationsystem.py
```

---

###  Run Tic Tac Toe AI

```bash
streamlit run tictactoe.py
```

---

##  Output

After running, open browser:

```
http://localhost:8501
```
